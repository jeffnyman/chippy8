ChipPy-8 is a `CHIP-8 <https://en.wikipedia.org/wiki/CHIP-8>`_ emulator and interpreter.

Usage
-----

.. code-block:: console

    $ chippy8 [OPTIONS]

.. option:: -d

    Provide debug logging as ChipPy-8 executes.

.. option:: -i

    Provide information logging as ChipPy-8 executes.

    Note that running logging with just `-i` shows only info.
    Running logging with `-d` shows info and debug.

.. option:: -v, --version

    Display the version and exit.

.. option:: -h, --help

    Display information about ChipPy-8 usage and arguments.
